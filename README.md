# Data analyst course projects

Данные проекты были выполнены в ходе обучения в Яндекс.Практикуме профессии "Аналитик данных":


| Название       | Описание  | Библиотеки            |
| :-------------------:|:-------------------: |:---------------------------:|
| Анализ рынка недвижимости |В нашем распоряжении архив объявлений о продаже недвижимости. Нужно научиться определять рыночную стоимость квартир. | pandas, matplotlib
| Анализ рынка видеоигр |Из открытых источников доступны исторические данные о продажах игр, оценки пользователей и экспертов, жанры и платформы (например, Xbox или PlayStation). Нам нужно выявить определяющие успешность игры закономерности.                 | pandas, pyplot, seaborn, math, matplotlib
